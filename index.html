The linked and the minified version come as a single file, including a minimal AMD implementation called almond.js.

The last version loads the minified application via RequireJS, so at least two JavaScript files are loaded. The advantage is that a third JavaScript file containing ECMAScript 5 polyfills (es5-polyfills.js) is only loaded when needed, e.g. in Internet Explorer 8 and older. In all modern browsers, this obsolete code is not loaded at all. This shrinks the demo application from 10 kb to 5 kb, but of course needs require-min.js, which is 15 kb.
